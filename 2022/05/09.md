https://leetcode.cn/problems/di-string-match/
### 贪心，先保证局部，分割整体
```
class Solution {
    public int[] diStringMatch(String s) {
        int min = 0;
        int max = s.length();
        int[] ret = new int[s.length() + 1];
        for (int i = 0; i < s.length(); ++i) {
            if (s.charAt(i) == 'I') {
                ret[i] = min++;
            } else {
                 ret[i] = max--;
            }
        }
        ret[s.length()] = min; 
        return ret;
    }
}
```
